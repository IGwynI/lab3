### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для
последовательности

Получить список из минимальных цифр натуральных чисел, содержащихся в

Результат выводится как последовательность minDigits, которая определяется как исходная последовательность, к каждому элементу которой применили функцию minDigit (находит минимальную цифру в числе) с помощью Seq.map;

# Задание 3

## Задача 1

### Текст задачи

Вывести самое короткое название файла в указанном каталоге.

### Алгоритм решения

Ввод пути пользователем; 
Создается последовательность, которая определяется как последовательность имен файлов в каталоге, Directory.EnumerateFiles(path) является последовательность файлов в каталоге, из которой уже получаем имена с помощью Seq.map Path.GetFileName; 
Если каталог пуст - сообщаем об этом пользователю; 
Если каталог не пуст - находим самое короткое имя файла с помощью Seq.minBy, который последовательно применяет к каждому элементу последовательности функцию name.Length, после чего возвращает элемент последовательности с минимальной длинной; 
Вывод результата;
